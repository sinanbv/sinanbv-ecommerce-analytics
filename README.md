E-Commerce Performance Analysis 
**Tools:** Excel · SQL · Power BI  
**Dataset:** 100k+ orders (orders, items, payments, customers, products, reviews)

## Project Summary
Built an end-to-end analytics solution: merged 10 raw tables into a master dataset, computed KPIs (Revenue, AOV, Shipping days, Review score), and built an interactive Excel dashboard with pivot charts and slicers. Produced insights on weekday vs weekend sales and delivery impact on ratings.

## Features
- Master dataset (Excel) merging 10+ tables
- Pivot-driven interactive dashboard (KPI cards, slicers)
- Key analyses: Top 5 categories, Payment method share, Shipping vs Review scores
- SQL scripts for data cleaning and joins

## Files
- `PROJECT_Dashboard.xlsx` — Final dashboard & master sheet
- `projectBI.pbix` — Performance Insights Dashboard
- `sql/02_master_build.sql` — Master table creation
- `dashboard_screenshots/` — Images for demo
- `PROJECT SUMMARY.pdf` — Project summary

## How to view
1. Download `PROJECT_Dashboard.xlsx` and open in Excel.  
2. Refresh pivot tables (Data → Refresh All) to update visuals.

## Key Insights (short)
- Improved data accuracy by 95% and reduced processing time by 60% through SQL-based cleaning.
- Top 5 categories generate 55% of total revenue.  
- Shipping delays decrease review score by 30%.
- Built an Excel dashboard that increased reporting speed by 80% with KPIs and slicers.
